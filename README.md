# Brute-Force-Attack-OWASP-Bricks

# Overview
This Activity demonstrates brute force attack on an intentionally vulnerable web page (OWASP BRICKS) using burpsuite and hydra.
The objective is to understand the weakness and mitigation techniques.

# Tools Used:
1. Kali Linux 
2. Burp Suite
3. Hydra

# Target:
OWASP Bricks (local lab environment)
